Palindrome Tests
================

To run the palindrome tests:

1. Navigate to the directory containing the `run_tests.exe` file.
2. Double-click the `run_tests.exe` file.
3. The tests will automatically run and output the results to the console.

Note: If you encounter any issues with running the tests, please ensure that you have the latest version of Python 3 installed along with the necessary dependencies (PyInstaller).
